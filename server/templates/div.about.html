About this game

Around The World is an online game experiment built by Peter Bengtsson.

The inspiration for this game was a game I used to play when I was a kid called Backpacker that I loved because it combined both fun and learning something. This game doesn't try to copy that game but is definitely inspired by it.

My job is a Web Developer at Mozilla but this game is my weekend project that I work on to learn how to build a fun and interactive game. Because it's something I do for fun I only have time to add the features I personally find most enjoying to play and to develop.

All questions I write myself from reading Wikipedia articles or just from stuff I happen to know. My goal is to make it possible to have people playing the game to contribute questions and pictures.
